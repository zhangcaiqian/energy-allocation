# 留白 — Check-in 问题模板池

> 共 80 条问题，按 4 个时间段分组。每次 check-in 从对应时间段随机选取一条，避免连续重复。

---

## 使用说明

1. 每次 check-in 时，根据当前时间匹配对应时间段
2. 从该时间段的问题池中随机选取一条（排除最近 3 次已用的问题）
3. 用户看到问题后，选择 4 级精力状态回答
4. 后续可用 LLM 基于这些模板生成变体，进一步丰富题库

**精力状态选项**（所有问题通用）：
- 🟢 精力充沛（HIGH = 1.0）
- 🟡 状态尚可（MEDIUM = 0.65）
- 🟠 有点疲惫（LOW = 0.35）
- 🔴 快没电了（EXHAUSTED = 0.1）

## 数据结构参考

```typescript
interface CheckInQuestion {
  id: string          // 如 "M01", "N15", "E08", "L20"
  period: 'morning' | 'noon' | 'evening' | 'night'
  timeRange: {
    start: string     // "07:00"
    end: string       // "10:00"
  }
  text: string        // 问题文本
}

// 时间段定义
const PERIODS = {
  morning: { start: '07:00', end: '10:00' },
  noon:    { start: '11:30', end: '14:00' },
  evening: { start: '17:00', end: '20:00' },
  night:   { start: '21:00', end: '23:00' },
}

// 选取逻辑
function selectQuestion(
  period: Period,
  recentQuestionIds: string[]  // 最近 3 次已用的问题 ID
): CheckInQuestion {
  const pool = questions.filter(
    q => q.period === period && !recentQuestionIds.includes(q.id)
  )
  return pool[Math.floor(Math.random() * pool.length)]
}
```
